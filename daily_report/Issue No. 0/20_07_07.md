### 学习总结:

* 原来的时候，git命令只会：

* ```sh
  git pull
  
  git add ./
  git commit -m "123"
  git push
  ```

* 现在来看 git 还是收获巨大的。

* ```sh
  git merge
  git branch -f
  git cherry-pick
  git checkout
  git reset
  git rebase
  git revert HEAD
  git tag
  ......
  ```

* 等等命令，包括`git`的初步模型，都能够明白了。
